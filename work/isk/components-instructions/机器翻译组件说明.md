## 机器翻译

### 1. 简介

机器翻译是利用计算机将一种自然语言(源语言)转换为另一种自然语言(目标语言)的过程。本组件使用预训练的Transformer模型进行机器翻译，支持中英互译。

注：由于机器翻译模型运行较慢，建议限制输入的数据量。

### 2. 参数面板说明

+ 特征列：选择需要翻译的文本，单选。
+ 语言：分为中译英和英译中两种模式。

### 3. 输入输出桩说明

+ 输入桩：1个，连接数据对象。
+ 输出桩：1个，返回数据对象。

### 4. 节点报告

+ 返回包含翻译结果的数据对象，翻译结果的列名以 "_$translated$"  结尾。
